## 1. The Writing Experience

### Focus Mode / Zen Writing Environment
- Full-screen distraction-free mode with floating toolbar
- Ambient backgrounds + typewriter sounds
- Mood-adaptive UI color palette
- Flow state indicator

### AI Interview Mode
- AI asks questions instead of blank page
- Voice responses transcribed into structured stories
- Removes #1 barrier for non-writers

### Real-Time Writing Coach
- Passive voice highlighter, "show don't tell" detector
- Emotional arc visualization (live graph)
- Style comparison: "You write like Hemingway today"
